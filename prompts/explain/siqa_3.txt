# Question:
Quinn wanted to help me clean my room up because it was so messy. What will Quinn want to do next?
# Options:
(A) Eat messy snacks
(B) help out a friend
(C) Pick up the dirty clothes
# Answer:
The answer is: C
# Explanation:
Quinn want to clean the room up. Picking up the dirty clothes is one way to clean the room. Thus, quinn will want to pick up the dirty clothes next.

# Question:
Sydney had so much pent up emotion, they burst into tears at work. How would Sydney feel afterwards?
# Options:
(A) affected
(B) like they released their tension
(C) worse

The answer is: B

# Question:

# Options:
(A) scared of losing the cat
